Arizona Empowerment Scholarship Account (ESA) Analysis
This project explores the impact of district wealth on the participation in the Arizona Empowerment Scholarship Account (ESA) program. Below are the resources generated from the analysis:

Honors Thesis

The detailed analysis examining the effects of district wealth on ESA program participation. This comprehensive study leverages data from the Arizona Department of Education and employs OLS regression to understand the dynamics of participation across economic backgrounds.

View Honors Thesis

ESA Participation Model Slides

A presentation that summarizes the program's background, motivation, methodology, and key findings. It provides a visual overview of the statistical analysis and conclusions.

View Model Slides

ESA Analysis Notebook

The Jupyter Notebook contains the code used for data analysis and visualization. It outlines the empirical methodology, data processing, and regression analysis performed during the study.

Note: For GitHub Pages, we recommend converting the Jupyter Notebook to HTML or PDF for easier viewing.
